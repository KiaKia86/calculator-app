    How to Use the Calculator

    Basic Operations:
    Use +, -, *, / for addition, subtraction, multiplication, and division.
    Example: 2 + 3 * 4 = → 14

    Scientific Functions:
    Use sin, cos, tan, log, ln, sqrt, ^, !.
    Example: sin(30) = → 0.5 (in degree mode)

    Constants:
    Use π for pi and e for Euler's number.
    Example: π = → 3.141592653589793

    Memory Functions:
    Use MC, MR, MS, M+, M- for memory operations.
    Example: 5 MS, MR → 5

    Error Handling:
    Invalid inputs will display an error message.
    Example: 2 / 0 = → Error: Division by zero
